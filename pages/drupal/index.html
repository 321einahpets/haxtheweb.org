Getting HAX setup in BackdropCMS

  1. Go to the HAX project page on drupal.org
    • Be aware that the Drupal 8 version of HAX does not require the web components module while Drupal 6 and 7 do
  2. Download the module for your platform and follow the directions found in the README.md
    • Drupal 6 and 7 have these files in theÂ CopyThisStuff folder found in the web components module so make sure you get that!
  3. Enable the HAX module
    • Web components module too if in Drupal 6 / 7
  4. Go to a node in your site, you should see a tab that says "HAX mode"
  5. Enjoy HAXing your Drupal site!

Special note on Drupal 7

In order to get the upload capabilities in Drupal 7 you'll need to download the File Entity and RESTws modules as well
